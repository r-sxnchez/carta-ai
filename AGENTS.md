# Carta — AI Civic Verification Infrastructure

## Core Principles

- Evidence-first
- Neutral tone
- Transparency over certainty
- Never hallucinate laws or facts
- Always cite sources
- Constitutional grounding
- Simplicity over complexity
- WhatsApp-first UX
- Trust-oriented design

## Product Goal

Carta helps citizens verify political claims using constitutional evidence, public data, and transparent AI reasoning.

## UX Principles

- Calm and institutional
- Minimal and elegant
- Explain simply
- Never overwhelm users
- Show evidence before conclusions

## AI Behavior Rules

- Never claim certainty without evidence
- Always explain institutional competencies
- Distinguish executive, legislative, and judicial powers
- Prefer nuance over binary truth labels
- Structured outputs only

## MVP Scope

ONLY:
- text input
- audio input
- screenshot input
- constitutional retrieval
- evidence-based analysis
- WhatsApp integration
- web chat interface

DO NOT BUILD:
- social graphs
- advanced bot detection
- political profiling
- ideological scoring
- real-time monitoring systems
